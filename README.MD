# Objetivo
Implementação de uma solução blockchain local em rust

# Funcionalidades
- Criar blocos e cadeias
- Gerar o bloco principal
- Executar transações na blockchain
- Gerar novos blocos
- Ajustar dificuldade
- Ajustar Recompensa

## Executando
cargo run
